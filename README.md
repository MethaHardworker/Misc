# CSV Reports Generator

Скрипт для генерации отчетов из CSV-файлов с данными о сотрудниках. Программа обрабатывает данные о закрытых задачах и формирует различные аналитические отчеты.

## Основная функциональность

- Чтение и обработка CSV-файлов с данными сотрудников
- Формирование отчета по эффективности (performance) с расчетом средней эффективности по должностям
- Возможность добавления новых типов отчетов через наследование классов
- Вывод результатов в виде форматированных таблиц в консоль

## Использование

```bash
python reports.py --files employees1.csv employees2.csv --report performance
```
